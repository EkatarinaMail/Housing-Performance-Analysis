# Housing Association: Rent Arrears Performance Analysis
Project Overview: This project demonstrates data analysis techniques used to monitor rent arrears within a Housing Association. The goal is to identify high-risk tenants and regional trends to improve collection performance.
Key Skills Demonstrated:
•	Data Modeling: Linked tenant data, manager directories, and payment records using Nested VLOOKUP.
•	Data Aggregation: Calculated total arrears by region using SUMIFS for executive reporting.
•	Data Visualization: Created an interactive dashboard with automated charts.
•	Performance Monitoring: Implemented Conditional Formatting to highlight arrears exceeding 4 weeks (High Priority).
Tools Used: Microsoft Excel (Nested VLOOKUP, SUMIFS, and Data Visualization).
